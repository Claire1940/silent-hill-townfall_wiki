<Section icon={Sparkles} title="コアゲームプレイ：CRTVシステム">

『サイレントヒル2 リメイク』のような三人称視点とは異なり、『タウンフォール』は純粋な一人称視点の体験です。この視点は、環境の閉塞感を強調し、パズルの触覚的な性質を際立たせるために設計されています。そして、ゲームプレイの中心となるのが「CRTV（ブラウン管バルブ）」ポケットテレビです。

<Steps>
  <Step num="1" title="信号の探知">
    セント・アメリアを探索していると、CRTVからノイズが発生します。手動でダイヤルを調整して正しい周波数を見つけることで、隠された環境のヒントが明らかになります。
  </Step>
  <Step num="2" title="敵の追跡">
    CRTVは従来のラジオが進化したものとして機能します。「裏世界（アザーワールド）」のクリーチャーの存在を感知し、脅威が近づくにつれて画面のノイズが激しくなります。
  </Step>
  <Step num="3" title="ステルスと回避">
    サイモンは兵士ではありません。「覗き込み」アクションを駆使して角の先を確認しましょう。敵に見つかった場合は、視線を切り、環境に隠れることが最善の選択となることが多いです。
  </Step>
  <Step num="4" title="ナラティブパズル">
    CRTVの信号を現実世界のオブジェクトに重ね合わせることでパズルを解きます。この「シグナル・シフティング」により、肉眼では見えない隠されたテキストや道が浮かび上がります。
  </Step>
</Steps>

<Callout type="warning" title="戦闘に関する注意">
近接武器や銃器も登場しますが、弾薬は極めて希少です。戦闘は最終手段として設計されており、サイモンを無防備で疲弊した状態に追い込みます。
</Callout>

</Section>

<Section icon={Shield} title="PCシステム要件と技術仕様">

**「サイレントヒル タウンフォール」の予約**を検討しているPCプレイヤーにとって、ハードウェア要件は本作の高品質なライティングや霧のエフェクトを反映したものとなっています。現実世界と裏世界のシームレスな移行を実現するため、SSDとWindows 11の使用が推奨されています。

| コンポーネント | 最小 (1080p/30fps) | 推奨 (4K/30fps) |
| :--- | :--- | :--- |
| **OS** | Windows 11 x64 | Windows 11 x64 |
| **プロセッサ** | Intel i7-8700K / Ryzen 5 3600 | Intel i7-9700K / Ryzen 7 5700X |
| **メモリ** | 16 GB RAM | 32 GB RAM |
| **グラフィックス** | RTX 2060 Super / RX 6600 | RTX 3080 / RX 7800 XT |
| **DirectX** | バージョン 12 | バージョン 12 |
| **ストレージ** | 75 GB SSD | 75 GB NVMe SSD |

<Callout type="tip" title="パフォーマンス向上のために">

</Callout>
