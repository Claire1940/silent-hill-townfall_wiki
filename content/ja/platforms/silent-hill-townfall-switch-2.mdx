<Section icon={Cpu} title="技術的実現可能性：Switch 2でタウンフォールは動作するか？">

**サイレントヒル タウンフォール Switch 2** への移植の可能性を理解するには、確定しているPCシステム要件を確認する必要があります。開発の Screen Burn は、1996年のスコットランドの「ローファイながらも高精細」な美学を表現するために、高解像度テクスチャと高度なライティングを駆使しています。SSDの使用は必須要件となっており、これは任天堂の次世代機の噂されているスペックと一致します。

<CardGrid cols={2}>
  <Card title="PC最小スペック">
    - **GPU**: RTX 2060 Super / RX 6600
    - **RAM**: 16 GB
    - **ストレージ**: 75 GB SSD 必須
    - **ターゲット**: 1080p / 30fps (Low設定)
  </Card>
  <Card title="PC推奨スペック">
    - **GPU**: RTX 3080 / RX 7800 XT
    - **RAM**: 32 GB
    - **ストレージ**: 75 GB NVMe
    - **ターゲット**: 4K / 30fps (High設定 / アップスケーリング使用)
  </Card>
</CardGrid>

噂されている Switch 2 のスペックは大幅なパワーアップを示唆しており、DLSS（Deep Learning Super Sampling）のサポートも期待されています。これは、ブラウン管のノイズや霧をシミュレートするために重いポストプロセス効果を使用する本作にとって非常に重要です。もし Switch 2 がエントリーレベルの現代的なPCと同等のパフォーマンスを実現できるのであれば、初代 Switch よりも移植の可能性は格段に高まります。

| コンポーネント | PC最小要件 | Switch 2の噂 | 互換性 |
| :--- | :--- | :--- | :--- |
| **メモリ** | 16 GB RAM | 12 GB LPDDR5X | 調整が必要 |
| **ストレージ** | SSD 必須 | UFS 3.1 / NVMe | 一致する可能性大 |
| **機能** | DirectX 12 | Ray Reconstruction | 可能 |
| **アップスケーリング** | DLSS / FSR / TSR | DLSS 3.1 / 3.5 | 非常に相性が良い |

<Callout type="tip" title="開発者の洞察">
Screen Burn（旧 No Code）は、物語重視のゲームを様々なプラットフォームに最適化してきた実績がありますが、彼らが重視する「触覚的」なインターフェースには、正確な入力と高解像度のテキストレンダリングが必要不可欠です。

<Section icon={Swords} title="ゲームプレイメカニクス：CRTVとサバイバル">

『サイレントヒル タウンフォール』はシリーズを一人称視点へと移行させ、サイコホラーの没入感を高めています。敵を検知するために単純なラジオを使用していた過去作とは異なり、本作では **CRTV（ポータブルテレビ）** が導入されます。このデバイスは主要なサバイバルツールとして機能し、主人公サイモン・オーデルは不安定なアナログ信号を調整することで、隠れた脅威や物語の断片を解き明かしていきます。

<Steps>
  <Step num="1" title="シグナルの調整">
    CRTVを使って周波数をスキャンします。クリアな信号は、セント・アメリアの住民の居場所や、あなたを追う「裏世界」の存在を明らかにします。
  </Step>
  <Step num="2" title="環境の観察">
    一人称視点により、触覚的なパズル解きが可能になります。1996年当時のテクノロジーに隠された手がかりを探すため、オブジェクトを物理的に操作する必要があります。
  </Step>
  <Step num="3" title="回避とステルス">
    戦闘は状況に応じたものになります。「覗き込み」メカニクスを使って角の先を確認しましょう。多くの場合、霧の中で敵が見える前に、CRTVが脅威の接近を知らせてくれます。
  </Step>
  <Step num="4" title="リソース管理">
    道具は慎重に管理してください。近接武器であれ、希少な銃器であれ、セント・アメリアでのあらゆる遭遇には重大な結果が伴います。
  </Step>
</Steps>

ゲームプレイのループは「アクションよりもまず観察」を強調しています。プレイヤーは環境を読み解きながら、寒々しく孤立したセント・アメリアの島を探索しなければなりません。CRTVはクラシックなラジオの進化形として機能し、主人公の精神状態や町の変容する現実を視覚的なノイズと音の合図で表現します。

| メカニクス | 従来のサイレントヒル | サイレントヒル タウンフォール |
| :--- | :--- | :--- |
| **視点** | 固定 / 三人称 | 一人称 |
| **脅威の検知** | ポケットラジオ (音声) | CRTV (音声/視覚ノイズ) |
| **舞台** | アメリカ (メイン州など) | スコットランド (島) |
| **戦闘** | 直接的 / ぎこちない | 戦術的 / 回避重視 |

<Callout type="success" title="サバイバルのヒント">
新しい建物に入るときや濃い霧の中に踏み出すときは、常にCRTVを構えてください。信号の乱れは、追跡が始まる前に得られる唯一の警告であることが多いです。
</Callout>

<Section icon={Target} title="ストーリーと舞台：1996年スコットランド、セント・アメリア">

物語は、島にある町セント・アメリアに「事態を収拾する」ために呼び戻された男、サイモン・オーデルを追います。1996年という時代設定とスコットランド沿岸の孤立した環境を活かし、シリーズの中でも独特な雰囲気を醸し出しています。ストーリーは喪失、記憶、そして過去の行動の報いといったテーマを探求し、それらすべてがCRTVの不安定な信号を通して描写されます。

**主要な物語要素:**
- **サイモン・オーデル**: 曇った過去を持ち、町のクリニックと何らかの繋がりを持つ主人公。
- **ゾーイ**: セント・アメリアの看護師で、不可解な導きと物語の背景をプレイヤーに提供します。
- **霧**: 単なる視覚効果ではなく、セント・アメリアの霧は表面化しつつある「裏世界」の断片を隠しています。
- **因果応報**: プレイヤーの選択と探索の深さが、複数のエンディングのどれに到達するかに直接影響します。
